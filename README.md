# Gradebook Analyzer
A simple Python gradebook program for quick student grade analysis.
Enter student names and marks manually
Calculates average, median, highest and lowest scores
Assigns grades (A/B/C/D/F) based on marks
Shows how many students achieved each grade
Filters pass/fail students using a 40-mark threshold
Prints clear tables and summaries in the terminal
